# Modeles3D

Modèles 3D en *.stl à imprimer en impression 3D (draps, pièces techniques, art fractal).

## Objectifs

Divertissement et curiosité pour l'impression 3D.

## Utilisations

Logiciel de modélisation : [Blender](https://www.blender.org)
